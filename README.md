# React Context API Demo

A very simple demonstration of a site using React's Context API.

## Getting started

### Prerequisites

- [Node.js](https://nodejs.org/)
- A brain

### Up-and-running

```bash
npm start
```

- [Create React App](https://github.com/facebook/create-react-app)
